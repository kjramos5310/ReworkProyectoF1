LABORATORIO

  • El objetivo de este laboratorio es desarrollar una página web atractiva y efectiva que combine técnicas de scrollreveal y landingpage para captar la atención del usuario, presentar información relevante sobre Fórmula 1 de manera atractiva y guiarle hacia una acción específica.
  • ¿Qué es un Landingpage?
  • Es una página web independiente diseñada con un objetivo específico en mente, generalmente convertir a los visitantes en clientes potenciales o clientes.
  • ¿Qué es un ScrollRevael?
  • Es una biblioteca JavaScript que te permite animar elementos HTML a medida que se desplazan por la pantalla.
